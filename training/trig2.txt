────────────────────────────────────────
PURPOSE OF THIS TRAINING (TRIGONOMETRY)
────────────────────────────────────────

The AI receives a natural-language description of a trigonometric function or expression and must output the correct symbolic trigonometric expression.

The AI must:
• Identify trigonometric functions
• Correctly map verbal phrasing to sin, cos, tan, etc.
• Preserve angles, variables, constants, and grouping
• Handle common trig modifiers (amplitude, frequency, shifts)
• Respect parentheses and order of operations

The AI must NOT:
• Simplify trig expressions unless explicitly asked
• Apply identities automatically
• Convert between degrees and radians unless stated
• Explain reasoning
• Infer missing parameters

────────────────────────────────────────
SUPPORTED TRIG FUNCTIONS
────────────────────────────────────────

The AI should recognize and translate the following functions:

• sine → sin
• cosine → cos
• tangent → tan
• cosecant → csc
• secant → sec
• cotangent → cot

Inverse trig (only if explicitly stated):
• arcsine → arcsin or sin⁻¹
• arccosine → arccos or cos⁻¹
• arctangent → arctan or tan⁻¹

────────────────────────────────────────
ANGLE VARIABLES AND DEFAULTS
────────────────────────────────────────

If an angle variable is not specified, assume x.

Examples:
“the sine of an angle” → sin(x)
“cosine of a number” → cos(x)
“tan of the input value” → tan(x)

If a variable is named, preserve it exactly.

Examples:
“the sine of θ” → sin(θ)
“cosine of t” → cos(t)

────────────────────────────────────────
BASIC TRIG TRANSLATIONS
────────────────────────────────────────

Examples:

Description:
“the sine of x”
Expression:
sin(x)

Description:
“cosine x”
Expression:
cos(x)

Description:
“tangent of the angle t”
Expression:
tan(t)

Description:
“the cosine of x squared”
Expression:
cos(x^2)

Description:
“the square of sine x”
Expression:
(sin(x))^2

IMPORTANT:
Distinguish between:
• sin(x^2)
• (sin(x))^2

Wording usually determines grouping.

────────────────────────────────────────
COMMON LANGUAGE PITFALLS
────────────────────────────────────────

Pay careful attention to wording:

“sine of x squared” → sin(x^2)
“square of sine x” → (sin(x))^2

“cosine of 2x” → cos(2x)
“two cosine x” → 2cos(x)

Order and grouping matter.

FUNCTION NOTATION
────────────────────────────────────────

Examples:

Description:

Expression:

Description:
“g of t equals cosine of t minus 1”
Expression:

────────────────────────────────────────
WHAT NOT TO DO (TRIG)
────────────────────────────────────────

Do not:
• Apply identities (e.g., sin²x + cos²x = 1)
• Simplify fractions
• Convert tan to sin/cos
• Assume radians or degrees unless stated
• Add domain restrictions

Example:
Input: “sine x divided by cosine x”
Correct: sin(x)/cos(x)
Incorrect: tan(x)
